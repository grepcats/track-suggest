C#/.NET

C# is a general-purpose, object-oriented programming language. Prospective students who personally prefer a Windows environment or are interested in developing applications for Windows may be interested in this track. C#/.NET isn't just for Windows, however. Recently Microsoft has open-sourced C# & its platform so that you can run and develop with it on the Mac & Linux operating systems.

If you're interested in creating games, you can even use C# scripting for Unity.

For more information on C# & .NET, check out the following resources

  • Wikipedia entry for C#
  • Wikipedia entry for .NET
  • Introduction to the C# language and the .NET Framework
  • C# at Epicodus

Java/Android

Java is one of the most popular programming languages and would be an attractive addition to your resume. It's versitile - you can see it used in any number of applications and in many different kinds of organizations. It is also the language of choice for Android mobile app development, making it an important language to learn if you are interested in developing for mobile.

Android is a mobile operating system developed by Google. The majority of mobile users are using Android devices, making this track a solid option for employment opportunities.

Ruby/Rails

Ruby is the language of choice for building dynamic, interactive web applications. It's a popular language due to its easy-to-use Rails framework and natural, elegant syntax. It's often used by smaller companies and startups.

PHP/Drupal

PHP is one of the most popular programming languages. It's most often used for database-backed websites with lots of content. Content management systems like WordPress and Drupal use PHP as well. In learning PHP, you will be learning to collect form data and generate web page content dynamically.

As mentioned before, Drupal is an open-source content-management framework written in PHP. It's used for many different kinds of sites requiring content management, such as blogs, government sites, and library content databases
